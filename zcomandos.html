Tabela de Tags HTML e Suas Descrições

Tag Descrição
<a> Define um hyperlink (link).
<h1> Título de maior importância (cabeçalho de nível 1).
<h2> Título de segunda maior importância (cabeçalho de nível 2).
<h3> Título de terceira importância (cabeçalho de nível 3).
<h4> Título de quarta importância (cabeçalho de nível 4).
<h5> Título de quinta importância (cabeçalho de nível 5).
<h6> Título de menor importância (cabeçalho de nível 6).
<font> Define a fonte do texto (deprecado no HTML5).
width Atributo que define a largura de um elemento.
href Atributo que especifica o destino de um link <a>.
height Atributo que define a altura de um elemento.
<fieldset> Agrupa elementos de um formulário em um conjunto.
<td> Define uma célula de dados em uma tabela.
<tr> Define uma linha de tabela.
align Atributo para alinhar o conteúdo (esquerda, centro, direita).
<label> Define um rótulo para um controle de formulário.
<label> com checkbox Define uma caixa de seleção dentro de um rótulo de formulário.
<label> com radio Define um botão de rádio dentro de um rótulo de formulário.
<ul> Define uma lista não ordenada (com marcadores).
<ol> Define uma lista ordenada (com números).
<img> Insere uma imagem na página.
<video> Insere um vídeo na página.
<p> Define um parágrafo.
<strong> Define um texto importante (negrito).
<em> Define um texto em ênfase (itálico).
<u> Define um texto sublinhado (deprecado em HTML5).
<s> Define um texto riscado (strike-through).
<div> Define uma divisão ou seção em um documento.
<span> Define uma seção em linha (útil para estilização).
<br> Insere uma quebra de linha.
<form> Define um formulário HTML para entrada de usuário.
<input> Insere um campo de entrada de dados em um formulário.
<button> Insere um botão clicável.
